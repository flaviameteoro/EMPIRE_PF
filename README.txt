To check that you can compile, first modify the Makefile so that the compiler, compiler options and libraries are selected appropriately for your system.

To compile, simply type 
   	    make

This will executables in the subdirectory bin/

Next, edit the file model_specific.f90 to make it appropriate for your model.

See the documentation for more explanation. doc/html/index.html and/or doc/latex/refman.pdf
